Add the following CNAME record to your DNS configuration to verify ownership of {{ provider.custom_domain }}:

Type Name Value TTL

If you're using a domain registrar like GoDaddy, Namecheap, or Cloudflare, follow these steps:

  1. Log in to your domain registrar's website
  2. Find the DNS management section
  3. Add a new CNAME record with the values above
  4. Save your changes

Note: DNS changes may take up to 48 hours to propagate, though they're usually much faster.

Step 1: Add a CNAME record

Add the following CNAME record to your DNS configuration to point your domain to our service:

Type Name Value TTL
CNAME www {{ default_domain }} Auto or 3600

Step 2: Add a TXT record for verification

Add the following TXT record to verify domain ownership. This helps us confirm you control the domain.

Type Name Value TTL
TXT @ {{ verification_code }} Auto or 3600

If you're using a domain registrar like GoDaddy, Namecheap, or Cloudflare, follow these steps:

  1. Log in to your domain registrar's website
  2. Find the DNS management section
  3. Add a new CNAME record for 'www' pointing to {{ default_domain }}
  4. Add a new TXT record with the verification code above
  5. Save your changes

Note: DNS changes may take up to 48 hours to propagate, though they're usually much faster.

Step 3: Verify Domain

Once you've added both records, click the button below to verify your domain.
